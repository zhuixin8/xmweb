## ✨ 核心功能

### 📝 AI 内容创作
- **AI 一键成文**：给个标题/选题，自动生成结构完整的文章
- **热点聚合**：实时抓取全网热点，一键基于热点选题创作
- **去 AI 味**：内置「人味化」改写引擎，让 AI 文章读起来像真人写的，降低同质化
- **多模型支持**：DeepSeek、硅基流动、智谱、火山、Cloudflare、OpenAI 兼容接口任选，可填自己的密钥
- **智能配图**：自动检索高清配图，图文并茂

### 📤 多平台自动发布
- **一键分发**：一篇内容勾选多个平台，自动并行发布
- **平台适配**：自动处理各平台的标题、正文、封面、话题、可见性、排版差异
- **真实浏览器自动化**：模拟真人操作发布，稳定可靠
- **发布日志 & 失败重试**：每次发布全程留痕，失败可一键智能重试

### ⏰ 定时托管 & 矩阵运营
- **定时计划**：设定好时间，自动「抓热点 → AI 成文 → 自动发布」全流程无人值守
- **多账号矩阵**：管理多平台、多账号，集中调度
- **配额与团队**：支持账号体系、用量统计

## 🖥️ 两种使用方式

| | 网页端 | 桌面端（推荐） |
|---|---|---|
| **安装** | 打开网址即用 | 下载安装包 |
| **发布方式** | 云端自动发布 | 本地浏览器发布，**全程可见、更稳** |
| **适合** | 随时随地轻量使用 | 矩阵号 / 重度运营 |

> 桌面端登录账号时打开真实浏览器窗口扫码/手动登录，Cookie 安全保存在本地，发布过程肉眼可见、更安心。
